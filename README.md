# Exercise Timer App

### React-Native Expo + UI Design Neumorphism

### 기능

- 분 초 단위로 운동 시작 시간과 휴식시간을 설정

- progressbar를 통해 운동시간과 휴식시간 진행율 표시

- start / paused reset 기능

### 구현예정

- 세트 수를 포함해 전체를 반복할 수 있는 기능
- navigation을 이용해 기록화면에 운동 기록 리스트 구현
- UI개선
